删除数据

  • 使用 DELETE 语句从表中删除数据

    table_name指定要执行删除操作的表；“[WHERE ]”为可选参数，指定删除条件，如果没有WHERE子句，DELETE语句将删除表中的所有记录。

  • 使用 WHERE 子句删除指定的记录。

  • 在删除数据时，也有可能因为约束的影响，导致删除失败
  • 小结：DML操作默认情况下，执行完以后都会自动提交数据。
  • 如果希望执行完以后不自动提交数据，则需要使用 SET autocommit = FALSE.
